**Create `OhMyClaw/PDF/LMStudioClient.swift`:**

**Imports:** Foundation

**LMStudioError enum** (Error, Sendable):
- `.unreachable` — LM Studio not running / connection refused
- `.badResponse(statusCode: Int)` — non-200 HTTP status
- `.emptyResponse` — choices array empty or nil content
- `.decodingFailed(String)` — JSON decode error

**API Codable models** (all Sendable):

`ChatCompletionRequest: Encodable`:
- `let model: String`
- `let messages: [ChatMessage]`
- `let temperature: Double`
- `let max_tokens: Int`
- Nested `ChatMessage: Encodable` with `role: String`, `content: String`

`ChatCompletionResponse: Decodable`:
- `let choices: [Choice]`
- Nested `Choice: Decodable` with `message: Message`
- Nested `Message: Decodable` with `content: String`

`ClassificationResult: Decodable`:
- `let is_paper: Bool`

**LMStudioClient struct** (Sendable):
- Properties: `let baseURL: URL`, `let modelName: String`, `let timeout: TimeInterval`
- `init(port: Int, modelName: String, timeout: TimeInterval = 30)`:
  - Build `baseURL = URL(string: "http://localhost:\(port)/v1")!`
  - Store modelName and timeout

**Health check — `func isAvailable() async -> Bool`:**
- Build URL: `baseURL.appendingPathComponent("models")`
- Create URLRequest with `timeoutInterval = 5` (quick check)
- `try await URLSession.shared.data(for: request)`
- Return true if `HTTPURLResponse.statusCode == 200`, false on any error
- Catch all errors → return false (includes `URLError.cannotConnectToHost`)

**Classification — `func classify(text: String, metadata: PDFMetadata) async throws -> Bool`:**
- Build URL: `baseURL.appendingPathComponent("chat/completions")`
- System prompt (designed for binary classification):
  ```
  You are a document classifier. Your task is to determine whether a document \
  is a scientific/academic paper. Respond with ONLY a JSON object: {"is_paper": true} \
  or {"is_paper": false}. No other text.
  
  A scientific paper includes: peer-reviewed articles, preprints, conference papers, \
  theses, dissertations, and technical reports from academic or research institutions. \
  Papers may be in any language.
  
  If you are unsure, respond with {"is_paper": false}.
  ```
- User content: prepend metadata (title, author, subject) if available, then "Document text:\n{text}"
- Build `ChatCompletionRequest` with `temperature: 0.0`, `max_tokens: 20`
- Encode body via JSONEncoder
- Build URLRequest: POST, Content-Type application/json, httpBody, timeoutInterval = timeout
- `let (data, response) = try await URLSession.shared.data(for: request)`
- Guard HTTPURLResponse statusCode == 200, else throw `.badResponse(statusCode:)`
- Decode `ChatCompletionResponse`, guard `choices.first?.message.content`, else throw `.emptyResponse`
- Return `parseClassification(content)`

**Parse helper — `private func parseClassification(_ response: String) -> Bool`:**
Conservative: only return true when clearly positive.
- Trim + lowercase the response
- Try JSON decode as `ClassificationResult` first — return `is_paper` value
- Fallback: check for string patterns `"is_paper": true` or `"is_paper":true`
- Anything else: return false (conservative — better to miss a paper than misfile a receipt)

**Retry wrapper — `static func classifyWithRetry(text:metadata:client:maxRetries:) async -> Bool?`:**
- `let backoffSeconds: [UInt64] = [2, 4, 8]`
- Loop `0...maxRetries` (default maxRetries = 3, so 4 total attempts):
  - `try await client.classify(text:metadata:)` — return result on success
  - On error: log warning with attempt number and error
  - If not last attempt: `try? await Task.sleep(nanoseconds: backoffSeconds[attempt] * 1_000_000_000)`
- After all retries exhausted: return nil (caller treats nil as "leave in Downloads")

Key constraints:
- Use `Task.sleep` not `Thread.sleep` (cooperative suspension per pitfall #10)
- 30s timeout per request, 5s timeout for health check
- Temperature 0.0 for deterministic classification
- max_tokens 20 — only need `{"is_paper": true/false}`

**Create `OhMyClaw/PDF/PDFTask.swift`:**

**Imports:** Foundation

**PDFTask struct** (FileTask, Sendable):
- `let id = "pdf"`
- `let displayName = "PDF Classification"`
- `let isEnabled: Bool`
- Private dependencies: `identifier: PDFFileIdentifier`, `textExtractor: PDFTextExtractor`, `client: LMStudioClient`, `destinationPath: String`
- Init: accept all dependencies, set `isEnabled` from a parameter

**canHandle(file:) → Bool:**
- Delegate to `identifier.isRecognizedPDFFile(file)`

**process(file:) async throws → TaskResult:**

Step 1 — Extract text and metadata:
- `guard let extraction = textExtractor.extract(from: file)` else:
  - Log warning: "No text extractable from PDF" with filename context
  - Return `.skipped(reason: "No extractable text (image-only or password-protected)")`

Step 2 — Classify via LM Studio with retries:
- Call `LMStudioClient.classifyWithRetry(text: extraction.text, metadata: extraction.metadata, client: client, maxRetries: 3)`
- If result is nil (all retries failed):
  - Log error: "Classification failed after all retries" with filename
  - Return `.skipped(reason: "Classification failed — leaving in Downloads")`

Step 3 — Check classification result:
- `guard isPaper else`: return `.skipped(reason: "Not classified as a scientific paper")`

Step 4 — Move to ~/Documents/Papers:
- Expand tilde in `destinationPath`: `NSString(string: destinationPath).expandingTildeInPath`
- Build destination directory URL
- `try FileManager.default.createDirectory(at: destDir, withIntermediateDirectories: true)` — auto-create on first paper
- Build file destination: `destDir.appendingPathComponent(file.lastPathComponent)`
- Duplicate check: `guard !FileManager.default.fileExists(atPath: destination.path)` else return `.skipped(reason: "File already exists in Papers")`
- `try FileManager.default.moveItem(at: file, to: destination)` — move, not copy
- Log info: "Paper moved to ~/Documents/Papers"
- Return `.processed(action: "Classified as paper, moved to ~/Documents/Papers")`
