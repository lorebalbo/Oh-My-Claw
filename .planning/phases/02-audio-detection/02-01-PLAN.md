- `import UniformTypeIdentifiers`
- Static `supportedExtensions: Set<String>` containing: `"mp3", "m4a", "aac", "flac", "wav", "aiff", "aif"` (per user decision: MP3, AAC/M4A, FLAC, WAV, AIFF, ALAC — note `.m4a` covers both AAC and ALAC containers)
- `func isRecognizedAudioFile(_ url: URL) -> Bool`:

  2. Check extension is in `supportedExtensions` — reject if not
  3. Create `UTType(filenameExtension: ext)` and verify it `.conforms(to: .audio)` — reject if UTType is nil or doesn't conform
  4. Return true only if BOTH checks pass (per user decision: "strictest mode")
- Mark struct as `Sendable`

This is a pure synchronous check — no I/O, no async. Used as the first gate in the audio pipeline before any AVFoundation work.

  <action>
Create `OhMyClaw/Audio/AudioMetadataReader.swift` with two items:

**1. AudioMetadata struct:**
- Properties: `title: String?`, `artist: String?`, `album: String?`, `durationSeconds: Double`
- Sendable conformance
- Method `func hasRequiredFields(_ fields: [String]) -> Bool` that checks each field name ("title", "artist", "album") against the corresponding property — returns true only if ALL specified fields have non-nil values
- Method `func missingFields(_ fields: [String]) -> [String]` that returns the names of fields that are nil (for logging)

**2. AudioMetadataReader struct:**
- `import AVFoundation` and `import CoreMedia`
- Sendable conformance
- `func read(from url: URL) async throws -> AudioMetadata`:
  1. Create `AVURLAsset(url: url)`
  2. Load duration and metadata in one call: `let (duration, metadataItems) = try await asset.load(.duration, .metadata)`
  3. Convert duration: `CMTimeGetSeconds(duration)`
  4. Extract title via `AVMetadataItem.metadataItems(from: metadataItems, filteredByIdentifier: .commonIdentifierTitle).first`
  5. Extract artist via `.commonIdentifierArtist`
  6. Extract album via `.commonIdentifierAlbumName`
  7. Load string values with `try? await item?.load(.stringValue)` (async since macOS 12)
  8. Apply `.nonEmptyTrimmed` to each string value (trim whitespace, return nil if empty — per user decision "empty string counts as missing")
  9. Return `AudioMetadata` with all values

**3. String extension (if not already on URL+Extensions):**
- `var nonEmptyTrimmed: String?` — trims whitespace+newlines, returns nil if result is empty

Use the CURRENT async APIs only — never `asset.duration` or `item.stringValue` (deprecated synchronous properties). Per research: AVFoundation's common key space reads metadata across MP3 (ID3), M4A (iTunes atoms), FLAC (Vorbis), WAV (INFO), and AIFF formats without format-specific code.
