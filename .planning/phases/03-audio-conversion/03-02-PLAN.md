**QualityTier enum** (String, Comparable, CaseIterable, Codable, Sendable):

**QualityEvaluator struct** (Sendable):

<task type="auto">
  <name>Task 2: Extend AudioMetadata with format and bitrate</name>
  <files>OhMyClaw/Audio/AudioMetadataReader.swift</files>
  <action>
Modify `OhMyClaw/Audio/AudioMetadataReader.swift`:

**Extend AudioMetadata struct** — add two new fields:
- `let format: AudioFormat` — the detected audio format
- `let bitrateKbps: Int` — estimated bitrate in kbps (0 for lossless where not meaningful)

**Add readFormatInfo method** to AudioMetadataReader:
- `private func readFormatInfo(from url: URL, asset: AVURLAsset) async throws -> (AudioFormat, Int)`
- Load tracks: `let tracks = try await asset.load(.tracks)`
- Find first audio track: `tracks.first(where: { $0.mediaType == .audio })`
- If no audio track found: fall back to `AudioFormat.fromExtension(url.pathExtension)` with bitrate 0
- Load `estimatedDataRate` from track: `try await audioTrack.load(.estimatedDataRate)` — convert to kbps: `Int(estimatedDataRate / 1000)`
- Load `formatDescriptions` from track: `try await audioTrack.load(.formatDescriptions)`
- If formatDescriptions has entries, inspect first one via `CMAudioFormatDescriptionGetStreamBasicDescription`:
  - `kAudioFormatMPEGLayer3` → `.mp3`
  - `kAudioFormatMPEG4AAC` → `.aac`
  - `kAudioFormatAppleLossless` → `.alac` (CRITICAL: this handles M4A ambiguity — .m4a container with ALAC codec)
  - `kAudioFormatLinearPCM` → `.wav` or `.aiff` based on `url.pathExtension.lowercased()` (both use PCM, extension distinguishes)
  - `kAudioFormatFLAC` → `.flac`
  - Default: `AudioFormat.fromExtension(url.pathExtension)`
- For lossless formats (isLossless = true): force bitrateKbps to 0 (not meaningful, per research pitfall #3 — estimatedDataRate may return 0 or misleading values for lossless)

**Update read(from:) method:**
- After loading duration and metadata items, call `readFormatInfo(from: url, asset: asset)` to get `(format, bitrateKbps)`
- Pass format and bitrateKbps to AudioMetadata initializer

**Import additions:** Ensure `CoreMedia` and `CoreAudioTypes` are imported (for `CMAudioFormatDescriptionGetStreamBasicDescription`, `kAudioFormat*` constants). CoreMedia is already imported. CoreAudioTypes should be available via AVFoundation umbrella. If constants aren't found, import `AudioToolbox` instead.

**Key pitfall avoidance:**
- M4A ambiguity: always check formatDescriptions BEFORE defaulting .m4a to AAC — ALAC files in .m4a containers must be caught
- estimatedDataRate = 0 for lossless: use format.isLossless to bypass bitrate, treat as high quality
- All async loads use modern `load()` API, not deprecated synchronous properties
